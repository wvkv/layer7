## VPS vs VDS: Understanding the Difference

Layer7 clearly distinguishes between two infrastructure types:

**Shared Infrastructure (VPS):**
- Best for: Development servers, file storage, websites, email
- CPU Usage: 0-50% average, can spike to 100%
- Lower monthly cost
- Shared CPU resources with spike capacity

**Dedicated Infrastructure (VDS):**
- Best for: Game servers, cryptocurrency mining, Tor nodes, high-demand applications
- CPU Usage: Up to 100% permanently
- Guaranteed dedicated CPU cores
- Higher performance consistency
